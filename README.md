# Dexi Studio

Modern, responsive ve statik bir web sitesi tasarimi.

## Dosyalar

- `index.html` - Sayfa icerigi ve bolum yapisi
- `styles.css` - Responsive gorsel tasarim
- `script.js` - Mobil menu ve header etkilesimleri

## Calistirma

Projeyi yerelde acmak icin `index.html` dosyasini tarayicida acabilirsiniz.
Basit bir yerel sunucu ile denemek isterseniz:

```bash
python3 -m http.server 8000
```

Ardindan `http://localhost:8000` adresini ziyaret edin.
